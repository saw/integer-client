Generate a unique integer What is this?

Just to experiment with AWS Lamdba I made a Lambda function which generates a unique integer. Click the button above and this page will make an xhr to the aws endpoint that give you a unique number.

The unqiueness is guarunteed by dynamodb atomic writes . When you make a request for a number I make a dynamo update-item to add one to the saved value, sending the result.
